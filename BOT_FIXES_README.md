## 🎯 كيفية الاستخدام

### البوت الذكي (Chatbot):

#### الأسئلة المدعومة:

**1. التحيات:**
- "مرحبا" / "السلام عليكم" / "صباح الخير" / "مساء الخير"
- "hello" / "hi"

**2. البحث عن الوظائف:**
- "كيف ابحث عن وظيفة"
- "كيف اتقدم للوظيفة"
- "طريقة التقديم"

**3. التسجيل:**
- "كيف اسجل"
- "نسيت كلمة المرور"
- "تسجيل حساب"

**4. السيرة الذاتية:**
- "كيف ارفع السيرة الذاتية"
- "تحديث السيرة"

**5. للشركات:**
- "كيف اضيف وظيفة"
- "إضافة وظيفة"

**6. معلومات عامة:**
- "ما هي بوابة الوظائف"
- "خدماتكم"
- "help"

**7. الشكر والوداع:**
- "شكرا" / "شكراً"
- "وداعا" / "مع السلامة"

#### مثال على المحادثة:
```
المستخدم: مرحبا
البوت: مرحباً بك! كيف يمكنني مساعدتك في بوابة الوظائف؟

المستخدم: كيف ابحث عن وظيفة
البوت: يمكنك البحث عن الوظائف من خلال صفحة "تصفح الوظائف" 
       في القائمة الرئيسية. يمكنك التصفية حسب الموقع والصناعة ونوع الوظيفة.

المستخدم: كيف ارفع السيرة الذاتية
البوت: بعد تسجيل الدخول، اذهب إلى لوحة التحكم > سيرتي الذاتية > 
       اختر ملف PDF > ارفع.
```

---

### نظام التقييم (Rating):

#### كيفية الاستخدام:
1. انتقل إلى صفحة تفاصيل أي شركة
2. قم بتسجيل الدخول كباحث عن عمل (role_id = 1)
3. مرر للأسفل إلى قسم "مراجعة الشركة"
4. مرر الماوس على النجوم لرؤية التأثير
5. اضغط على النجمة المطلوبة (1-5)
6. سيظهر نموذج المراجعة تلقائياً
7. اكتب مراجعتك
8. اضغط "إرسال المراجعة"

#### التأثيرات البصرية:
- 🌟 **Hover Effect:** تلوين ذهبي + Zoom عند التحويم
- ⭐ **Selection:** تثبيت اللون الذهبي للنجوم المختارة
- 📝 **Form Display:** إظهار النموذج فقط بعد اختيار النجوم
- ✅ **Validation:** التحقق من اختيار النجوم وكتابة النص

## 🔧 ملاحظات تقنية

### البوت الذكي:

**الكود الرئيسي:**
```javascript
// قاعدة بيانات الردود
const autoResponses = {
  'مرحبا': 'مرحباً بك! كيف يمكنني مساعدتك...',
  'كيف ابحث عن وظيفة': 'يمكنك البحث عن الوظائف...',
  // ...المزيد
};

// البحث الذكي
if (autoResponses[userMessage]) {
  response = autoResponses[userMessage];
} else {
  // بحث جزئي عن الكلمات المفتاحية
  for (const [key, value] of Object.entries(autoResponses)) {
    if (userMessage.includes(key) || key.includes(userMessage)) {
      response = value;
      break;
    }
  }
}
```
